# INSTRUÇÕES DE EXECUÇÃO

Para rodar o projeto e executar todos os testes unitários, use os comandos a seguir no terminal, dentro da pasta raiz do projeto (onde está o pom.xml):

1.  **Acessar a pasta:**
    ```bash
    cd sistema-biblioteca
    ```

2.  **Compilar e Testar:**
    ```bash
    mvn clean install
    ```
    O resultado esperado é **[INFO] BUILD SUCCESS**, o que comprova que todos os testes foram aprovados.
